Here are a list of some of projects I've made. Most of them are open-source and can be improved. They are pets projects developped outside the professional context, on my free time.

Patroni

This is the 1st project of the Krampstudio, it's an online platform for lawyers, a virtual cabinet. A dedicated site is online patroni.fr (french site). You can also look at an example of a Patroni implementation : monavocat-enligne.fr.

jQuery UI L2L

A jQuery UI plugin that provides you a list-to-list widget, it enables you to connect multiples lists.

  • Demo sote
  • Github
  • jQuery Plugin Repository

Grunt Jsdoc

A Grunt plugin, the node.js build tool, that enables you to generate the JsDoc3 documentation of a JavaScript project.

  • Github
  • Grunt Plugin Repository
  • NPM Repository

Grunt Bower PostInst

A Grunt plugin, that runs post install commands on Bower components. It enables you to complete the automated tasks of installing client side dependencies with Grunt.

  • Github
  • Grunt Plugin Repository
  • NPM Repository

Chai-xml

A Chai plugin, the TDD/BDD assertion library. This plugin helps you to test and compare XML structures.

  • Github
  • NPM Repository

Aja.js

Aja.js stands for Asynchronous JavaScript And JSON(P) : Ajax without XML. It's a JavaScript library that helps you to make requests in the browser like a REST client.

  • Website
  • Github
  • NPM Repository
